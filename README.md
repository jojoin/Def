Def
===

### 可扩展的编程语言

![Def Logo](http://www.deflang.org/image/logo_small.png)

### 简介

Def 致力于提供：

1. 类、泛型等不损失效率的抽象
2. 重载、宏等可扩展的元编程
3. 符号绑定等自定义语法
4. 缩减代码长度提升编码和编译效率

通过编译强类型、完全类型推导、函数模板、嵌套函数、函数重载、类模板、扩展宏、模式匹配、 lambda表达式、闭包、操作符绑定等等一系列特征完成 Def 的目标：**将 C++ 的高效抽象和 Lisp 的元编程融为一体**。
